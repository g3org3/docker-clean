# docker-clean

## Install
```
$ npm i -g dockerclean
```

## Usage
```bash
$ docker-clean [options]

options:
  -v display version
  -s silent mode
  -a | --all (soon)
    by default now
  --con Remove stopped containers
  --img Remove unlabeled images
  --vol Remove orphaned volumes
```
